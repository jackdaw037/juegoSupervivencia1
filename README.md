# Juego Supervivencia RPG
Este es un juego de supervivencia para la práctica final RA3-RA4. El objetivo es gestionar recursos limitados para sobrevivir 10 días en un mundo apocalíptico.

## Introducción
Despiertas en una gasolinera abandonada con solo un machete y unas vendas. El mundo exterior está lleno de zombis.

## Juego
Introduce tu nombre de superviviente.
Elige entre las opciones del menú:
1. Ver estado: Comprueba tus estadísticas.
2. Ver inventario: Ver tus objetos.
3. Explorar: Explora zonas y lucha contra enemigos.
4. Dormir: Pasa al siguiente día y recupera energía.
5. Usar objeto: Cúrate o recupera fuerzas con lo que hayas encontrado.
0. Salir: Abandonar la partida.

## Organización
El programa utiliza (switch-case) para gestionar las acciones y (ArrayList) para controlar los enemigos, lugares e inventario para que haya diferentes aventuras en cada partida.
